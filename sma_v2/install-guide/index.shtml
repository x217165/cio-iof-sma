Installation guide

Ok, so you've ended up here. You want to know how to get it running. I'll tell you how...

First, get a chicken. Then, take 5 black candles and a pentagram. Light up the candles and put the chicken on the pentagram... nah, I'm only kidding. There's not much work to do, and no chickens, candles or pentagrams involved. There's no fee either.

First of all you need a bit of time. Don't rush this installation process unless you're an accustomed hacker and can figure out how everything works just by looking at the code behind everything. You'd also want to make sure you have some basic HTML skills, know what a frameset are and how they work.

Secondly you may want to drop by the download page. The files available there contains two small example web-sites using the menu, with instructions on how to make them work. If you want to you can read everything from there (the instructions are the same as the ones here), or you can read on.

This installation guide will go through the following items:

  • The frameset
  • Setting options
  • Making the menu
  • The icon menu
  • Tracking code
